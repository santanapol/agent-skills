# Compatibility Testing

## 📌 คืออะไร
ทดสอบบนแพลตฟอร์มที่หลากหลาย เช่น Browser ต่างกัน (Chrome, Safari, Firefox), OS ต่างกัน (Windows, macOS, iOS, Android) และ Device ต่างกัน

## 👥 ใครทำ
QA (Automated / Manual)

## 🛠️ เครื่องมือที่แนะนำ (Zero Platform Standard)
- **Automated:** `Playwright` — สามารถรัน Test ชุดเดียวซ้ำๆ บน Chromium, WebKit (Safari), และ Firefox รวมถึงจำลอง Mobile Viewport ได้ในตัว
- **Manual (ถ้าจำเป็น):** `BrowserStack` หรือเครื่องจริง

## 🔥 ความจำเป็น
⭐️ **Optional** — ขึ้นอยู่กับกลุ่ม User ที่ใช้งานระบบเป็นหลัก
