# UAT — User Acceptance Testing (ทดสอบเพื่อยอมรับระบบ)

## 📌 คืออะไร
ให้ "ผู้ใช้งานจริง" หรือ "ลูกค้า (Business)" ทดลองใช้ระบบ เพื่อยืนยันว่าตอบโจทย์ธุรกิจจริงๆ ไม่ใช่แค่ถูกตาม Spec ที่เขียนไว้

## 👥 ใครทำ
Product Owner, ลูกค้า, หรือ End-User ตัวแทน

## 🔥 ความจำเป็น
**Must-have** — ต้องผ่านด่านนี้ก่อน Deploy Production เสมอ
