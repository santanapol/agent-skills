# UAT Scenario: สร้างแคมเปญ Rolling Commission ประจำเดือน

**Business Goal:** แอดมินสามารถสร้างแคมเปญสำหรับให้โบนัสคืนยอดเสียกับผู้เล่นได้ด้วยตัวเอง เพื่อกระตุ้นยอดเล่นในแต่ละเดือน

## 📝 Pre-conditions
- แอดมินมีสิทธิ์ระดับ `Super Admin` หรือ `Marketing Manager`
- ระบบมียอดเล่น (Bet Log) ของผู้เล่นอยู่แล้วในเดือนปัจจุบันอย่างน้อย 1 คน
- ผู้ทดสอบมี Username ทดสอบฝั่งผู้เล่นเตรียมไว้ 1 บัญชี

## 🧑‍💻 Scenario 1: Happy Path (การทำงานปกติ)
**ขั้นตอนการทดสอบ:**
1. เข้าสู่ระบบ Backoffice ด้วยบัญชี Admin
2. ไปที่เมนู **Promotion > Rolling Commission**
3. กดปุ่ม **สร้างแคมเปญใหม่**
4. กรอกข้อมูล:
   - ชื่อแคมเปญ: `คืนยอดเสีย 5% เดือนนี้`
   - วันที่: เลือกวันที่ 1 ถึง สิ้นเดือนปัจจุบัน
   - Rate: `5%`
5. กดปุ่ม **บันทึกและ Publish**

**ผลลัพธ์ที่คาดหวัง (Acceptance Criteria):**
- [ ] แคมเปญแสดงในหน้ารวม (List Page) และมีสถานะเป็น `Active`
- [ ] ผู้ทดสอบสลับไปเข้าหน้าเว็บฝั่งผู้เล่นด้วยบัญชีทดสอบ
- [ ] ผู้ทดสอบมองเห็นแบนเนอร์หรือเมนูแคมเปญ `คืนยอดเสีย 5% เดือนนี้` แสดงขึ้นมา
- [ ] ยอดสะสมโบนัสตรงตามเป้าหมาย (หากมียอดเสียก่อนหน้า)

## 🧑‍💻 Scenario 2: Alternative Path (บันทึกเป็น Draft)
**ขั้นตอนการทดสอบ:**
1. ทำซ้ำขั้นตอนที่ 1-4 จาก Scenario 1
2. แทนที่จะกด Publish ให้กดปุ่ม **Save as Draft**

**ผลลัพธ์ที่คาดหวัง:**
- [ ] แคมเปญแสดงในหน้ารวม แต่สถานะเป็น `Draft`
- [ ] ผู้ใช้ฝั่งหน้าบ้าน (ผู้เล่น) จะ**ต้องมองไม่เห็น**แคมเปญนี้
- [ ] แอดมินสามารถกดเข้าไปแก้ไข (Edit) แคมเปญนี้ได้อีกครั้ง
