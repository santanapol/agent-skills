# Regression Testing

## 📌 คืออะไร
ทดสอบ Feature เดิมทั้งหมดซ้ำทุกครั้งที่มีการแก้โค้ดหรือเพิ่ม Feature ใหม่ เพื่อให้แน่ใจว่าของใหม่ "ไม่ทำของเก่าพัง"
**หมายเหตุ:** นี่คือ **Functional Testing** ไม่ใช่ Non-Functional เพราะทดสอบพฤติกรรมของระบบโดยตรง

## 👥 ใครทำ
QA (Automated เป็นหลัก เพื่อประหยัดเวลา)

## 🔥 ความจำเป็น
**Must-have** — ขาดไม่ได้สำหรับโปรเจกต์ที่อัปเดตบ่อย
